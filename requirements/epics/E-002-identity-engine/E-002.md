## Outcome

Noxis speaks with a consistent, opinionated identity from the very first interaction — and that identity never drifts regardless of user input, conversation length, or memory state. The personality layer (SOUL.md) is loaded as a fixed system prompt, the behavioral rules (AGENTS.md) govern response routing, and the tone mode adapts delivery style while preserving the core voice. Every response feels like it came from the same person.
